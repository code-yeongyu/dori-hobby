# Web UI Design System

## 1) Brand intent
Dark, operator-focused control panel for live AI gameplay supervision.

## 2) Color tokens
- `bg.base`: `#0d1117`
- `bg.panel`: `#161b22`
- `border.default`: `#30363d`
- `text.primary`: `#e6edf3`
- `text.muted`: `#8b949e`
- `accent.primary`: `#58a6ff`

## 3) Typography
- Font family: system UI stack
- `text.sm`: 12px
- `text.md`: 14px
- `text.lg`: 16px
- `text.xl`: 20px

## 4) Spacing scale
4px base: 4, 8, 12, 16, 20, 24.

## 5) Components
- `panel`: dark surface, 1px border, 8px radius.
- `status-pill`: compact bordered badge, accent or semantic tint.
- `chat-row`: sender label + message body with timestamp.

## 6) Motion
Subtle 120-200ms opacity/border-color transitions. No large transforms.

## 7) Accessibility
High contrast, keyboard-focus outlines with accent color, readable labels.
